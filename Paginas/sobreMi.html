Sobre mí

¡Hola! Soy Juan Pablo Rojas Contreras, un apasionado por la tecnología con una fuerte orientación hacia el desarrollo web y la creación de sistemas de software. Me encanta crear soluciones innovadoras y funcionales para usuarios y empresas mediante el uso de tecnologías modernas.

Estudié Ingeniería de Sistemas y actualmente estoy desarrollando proyectos y utilizando herramientas como Bootstrap 5 para construir interfaces de usuario responsivas y elegantes. Me gusta seguir aprendiendo y manteniéndome actualizado con las últimas tendencias en desarrollo y diseño web.

Mis habilidades:

  • Frontend: HTML, CSS, JavaScript, Bootstrap 5, Ajax, Sass.
  • Backend: Jquery, MySQL, Sql Server, .Net.
  • Versionado: Git, GitHub.
  • Otras herramientas: Visual Studio Code, Apache NetBeans, Visual Studio.

Me encanta enfrentar desafíos técnicos y siempre estoy buscando oportunidades para aplicar mis conocimientos a proyectos que resuelvan problemas reales. Mi enfoque es crear experiencias digitales atractivas y funcionales, asegurándome de que cada proyecto sea escalable y fácil de mantener.

¡Si necesitas más información sobre mí, no dudes en ponerte en contacto conmigo!
